# AGEP — Aplicación para Gestionar Emprendimiento Personal

PWA mobile-first para emprendedores. Responde en segundos: **¿Mi negocio es rentable hoy?**

## Stack

- **React + Vite** — Frontend
- **Tailwind CSS v4** — Estilos
- **Supabase** — Base de datos + Auth
- **Vercel** — Deploy

## Instalación local

```bash
# 1. Clonar el repositorio
git clone https://github.com/TU_USUARIO/agep.git
cd agep

# 2. Instalar dependencias
npm install

# 3. Correr en desarrollo
npm run dev
```

## Deploy en Vercel

1. Push a GitHub
2. Importar el repositorio en [vercel.com](https://vercel.com)
3. Deploy automático en cada push a `main`

## Módulos MVP

- ✅ Login / Registro (Supabase Auth)
- ✅ Onboarding — configuración inicial del negocio
- ✅ Dashboard — rentabilidad del mes
- ✅ Ingresos — registro de ventas
- ✅ Gastos — registro de egresos por categoría
- ✅ Inventario — materiales con alertas de stock
- ✅ Configuración — perfil, moneda, categorías
